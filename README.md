# PUZ BallPG 프로토타입

RPG 퀘스트 감각을 섞은 Godot 4 블록격파 프로토타입입니다.

- 등급별 형태가 다른 3D 퀘스트 블록
- 절차 생성 3D 보드 맵과 생성 이미지 기반 퀘스트/적 아이콘
- 퍼스펙티브 카메라 기반 보드게임 시점
- 화면 하단에 보이는 3인 파티 순환 턴
- 클릭/터치로 공 발사, 방향키로 발사 위치 이동, 점선 인디케이터로 각도 확인, Esc로 리셋
- 레벨이 오를 때마다 기존 블록이 아래로 내려오고 상단에 새 블록 행이 생성되는 블록격파 배치

Godot 4에서 `project.godot`을 열고 메인 씬을 실행하면 됩니다.

## 자산 메모

- 생성 이미지는 `assets/generated/`에 저장되어 있습니다.
- Blockbench 원본은 `assets/blockbench/quest_blocks.bbmodel`입니다.
- 현재 씬은 빠른 플레이 검증을 위해 3D 보드 맵과 Blockbench 블록 형태 언어를 절차 메시로 즉시 생성합니다.
