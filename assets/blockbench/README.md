# Blockbench Source Assets

`quest_blocks.bbmodel`은 프로토타입 블록 형태를 담은 편집 가능한 Blockbench 원본입니다.

- 사각 석재 퀘스트 블록
- 다이아 등급 블록
- 넓은 보스 블록
- 2D 퀘스트/적 이미지를 얹기 위한 작은 받침

현재 Godot 프로토타입은 바로 실행할 수 있도록 같은 형태를 절차 메시로 생성합니다. 비주얼 방향이 확정되면 이 `.bbmodel`을 Blockbench에서 열고 베벨/재질을 다듬은 뒤 GLB로 `assets/models/`에 내보내면 됩니다.
